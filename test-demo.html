ðŸš€ How to Test

1. Start the Application

# Backend (Terminal 1) cd backend && npm start # Frontend (Terminal 2) cd frontend && npm start

2. Test PowerSync ON

  1. Ensure PowerSync toggle is ON
  2. Create a new user - will save to both IndexedDB and MongoDB
  3. Check user list - should show "MongoDB + Local" badge
  4. Refresh page - user should persist (from MongoDB)

3. Test PowerSync OFF

  1. Toggle PowerSync to OFF
  2. Create a new user - will save to IndexedDB only
  3. Check user list - should show "Local Only" badge
  4. Refresh page - user should persist (from IndexedDB)

4. Test Offline Mode

  1. Disconnect internet
  2. Create/edit users - will save to IndexedDB only
  3. Reconnect internet
  4. Toggle PowerSync ON and click "Sync Now"
  5. All local changes will sync to MongoDB
